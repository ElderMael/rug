com.atomist.util.scalaparsing

CommonTypesParser

Related Doc: package scalaparsing

abstract class CommonTypesParser extends JavaTokenParsers with LazyLogging

Convenient support for parsers for different languages implemented using Scala parser combinators. Offers common tokens, consistent reserved word handling, comment and string handling, and productions for nested blocks.
