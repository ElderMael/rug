jsScalaHidingProxy

Related Docs: object jsScalaHidingProxy | package interop

class jsScalaHidingProxy extends AbstractJSProxy

Use this if you want to expose a structure in JavaScript with TypeScript-friendly arrays and null in place of Option. Simply uses reflection. Also, enables suppression of some methods. Exposes no-arg methods as properties but allows invocation with parameters. If you wish to expose a no-arg method as a function (because it has side effects etc) use
